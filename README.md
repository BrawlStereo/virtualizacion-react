# Optimización de virtualización en React

En este proyecto, hacemos una pequeña solución a problemas cotidianos en React como la renderización innecesaria de componentes hijos, o la asignación de valores clave a componentes con un enlace funcional. Más allá del código o la solución específica, nos quedamos con el conocimiento de cómo funciona React, tanto en el DOM, como en el aprovechamiento de funciones como memo o useCallback que le permiten al programa tener noción de lo que se está usando. Nuestras partes de colaboración vienen descritas en el documento [Documentación](Documentación.pdf).
